*************************
* UVisualize! Theme files
* For more information see the tutorials under http://cba.fro.at/uvisualize
*************************


Copy the three files into your theme directory (e.g. /mysite/wp-content/themes/twentyfifteen/)



	- single-uvis_playlist.php

Displays a single playlist. (e.g. www.mysite.com/playlist/my-playlist)



	- single-uvis_visualization.php

Displays a single visualization. (e.g. www.mysite.com/visualization/my-visualization)



	- archive-uvis_playlist.php

Displays an archive of playlists. ( e.g. www.mysite.com/playlist)



Update your permalink structure to make the pretty URLs (as listed above) accessible.
